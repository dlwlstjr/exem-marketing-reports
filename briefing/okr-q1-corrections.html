김유진 — O1 관련
KR "뉴스레터 오픈율 15%+"
뉴스레터 A/B 테스트 → 오픈율 15%+ 달성
뉴스레터 CTA 문구·위치 테스트 → 클릭률(CTR) 5%+ 달성
김유진 댓글: 오픈율 이미 40~60%. 클릭률이 더 의미 있는 지표. B2B 평균 CTR 2~3% 대비 5%+가 도전적이면서 현실적.
KR "A/B 테스트"
A/B 테스트 항목 미정 (제목? 발송 일시?)
CTA 문구·위치 테스트로 변경. 스티비 무료 요금제에서 링크별 클릭 추적 가능.
김유진 댓글 반영: CTR을 KR로 바꿨으니 CTA 테스트가 정합성에 맞음.
O1에 KR 추가
(해당 KR 없음)
고객성공사례 1건 완성 (케이뱅크) — 김유진(전담), 정진아(영상 디자인)
임수현 댓글 + 팀장 확인: 주요 목표인데 분류가 없었음. BOFU 콘텐츠로서 O1에 직접 부합. 김유진이 기획·콘텐츠 전담.
임수현 — O2 관련
O2 범위
AWS Summit 중심, 6월 내부 세미나 미기재
AWS Summit(5월, 외부 전시박람회) + 6월 MS-SQL 내부 세미나 모두 포함. KR 추가: "6월 내부 세미나 리드 수집 + 사후 콘텐츠"
임수현 댓글: 2Q 행사가 2건(외부 1 + 내부 1)인데 하나만 반영됨.
KR "콘텐츠 3건" + 역할 구분
콘텐츠 3건 (채널 구분 없음) — 임수현(기획 & 블로그)
사후 콘텐츠 1세트 = 블로그 1(김유진) + SNS 2(임수현)
임수현 댓글 + 팀장 확인: 블로그는 김유진 담당, 임수현은 SNS(인스타/링크드인) 담당. 역할 구분 명확화.
KR "SQL Tuning + AWS"
SQL Tuning + AWS (용어 혼선)
SQL Tuning = 3월 완료된 내부 세미나. AWS = 5월 외부 행사. 별개 행사임을 명시. 6월 MS-SQL 내부 세미나는 별도 KR로 추가.
임수현 댓글 + 팀장 확인: SQL Tuning(1Q 완료)과 6월 MS-SQL 세미나(2Q 예정)는 다른 행사. 혼동 방지.

팀장 확인: 은빈님은 타팀 TF 중. 솔루션맵은 완성 상태이므로 2Q KR에서 제거.
2Q OKR 구조 논의 주간회의 안건
정진아님이 제기한 구조적 이슈. 2Q OKR 확정 전에 팀 차원에서 결정이 필요해요.
타부서 지원을 OKR에 포함할 것인가?
현재 제안
70% OKR + 30% 스팟 버퍼. 타부서 요청은 OKR 밖의 '운영 업무'로 분리.
정진아 의견
BX팀이 사내에서 담당하는 역할상, 타부서 지원 자체를 팀 OKR 중 하나로 잡는 방식도 검토 필요. 디자이너가 일정·우선순위를 조율할 수 있는 경우가 매우 드물고, 타부서 니즈에 맞춰 진행됨.
정진아 댓글 3건(4/6) — 타부서 대응이 BX팀 본업 중 하나인데 OKR에서 빠지면 성과로 인정받지 못하는 구조적 문제
팀장 판단 필요
A안: 현행 유지 (70/30 버퍼). 타부서 지원은 주간 성과에서 별도 트래킹.
B안: 팀 O4 신설 "타부서 디자인 요청 대응 시간 단축" — KR: 요청→완료 평균 리드타임 측정, 요청 양식 정착률 등.
C안: O3에 통합. "디자인 자산 정리 + 타부서 셀프서비스 가능하게" — 가이드 문서가 곧 타부서 지원 감소로 연결.
